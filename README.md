# Youtube Clone

React를 이용해서 Youtube 클론</br></br>
메인 화면은 유튜브 인기영상 리스트를 보여주고 영상을 클릭하면 해당 영상을 볼 수 있다.</br>
키워드를 검색하면 키워드에 해당하는 영상 리스트를 보여준다.</br>
영상을 클릭하면 상세페이지를 보여준다.
